## Convert all figures to TIFF for PLOS submission

PLOS ONE requires 300 DPI TIFF with LZW compression. The `convert_figures_to_tiff.py`
script converts every PNG in the run's figures/ folder to TIFF:

```bash
python convert_figures_to_tiff.py runs/<RUN_ID>/figures/
```

Figure 14 is already generated as TIFF natively by `make_figure_14.py`.
